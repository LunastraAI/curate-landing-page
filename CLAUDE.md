# Curate - AI Stylist Landing Page

## Project Overview
A landing page for "Curate" - an AI-powered styling mobile app that helps users generate outfit recommendations and shop the items.

## Goals
1. **Primary**: Showcase the app's luxury-meets-creativity vibe
2. **Secondary**: Drive iOS app downloads

## Brand Identity
- **Name**: Curate
- **Tagline**: Your AI Stylist
- **Tone**: Luxury, creative, approachable, modern

## Design Language (from app)
- **Background**: Clean white/cream (#FAFAFA or similar)
- **Primary Text**: Black/dark gray for headlines
- **Secondary Text**: Medium gray for body text
- **Accent Color**: Warm gold/tan/champagne (approximately #C9A962 or similar)
- **Typography**: Bold, modern sans-serif for headlines; clean sans-serif for body
- **UI Style**: Minimalist, lots of whitespace, rounded elements (pill buttons, rounded inputs)

## App Features to Highlight
- AI-powered outfit generation based on occasions
- Quick picks for common events (casual weekend, job interview, first date, etc.)
- Chat-based interface for describing styling needs
- Ability to save and shop curated looks

## Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Deployment**: Vercel
- **Fonts**: next/font (for optimized loading)
- **Images**: next/image (for optimization)

### Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Run production build
- `npm run lint` - Run ESLint

## Key Sections (Proposed)
- Hero section with app preview
